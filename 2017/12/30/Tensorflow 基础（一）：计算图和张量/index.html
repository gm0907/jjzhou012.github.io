张量

tensorflow中所有数据以张量形式表示。
从功能角度，张量可以理解为多维数组。零阶张量表示标量（scalar），也就是一个数；一阶张量表示向量（vector），也就是一维数组；第n阶张量可以理解为n维数组。

但是，张量在tensorflow中的实现不是采用数组形式，它只是对tf运算结果的引用，并未保存真正的数字，而是保存得到数字的计算过程。
以向量加法为例：

Tensor("add_2:0", shape=(2,), dtype=float32)

运行以上代码并不会得到运算结果，而是得到对结果的一个引用。Tensorflow的计算结果不是一个具体数字，而是一个张量结构。
一个张量中主要保存了3个属性： 名字（name）、维度（shape）和类型（type）。

  • name 属性不仅是一个张量的唯一标识符，也给出了一个张量是如何运算出来的。

Tensorflow 的计算都可以通过计算图的模型来建立，计算图上的每一个节点代表了一个计算，计算的结果保存于张量中。因此，张量对应于计算图上节点所代表的计算结果。张量的命名可以通过“node：src_output”的形式给出，其中，node 为节点名称，src_output 表示当前张量来自节点的第几个输出。
例如，上述代码段的Print结果“add：0”说明了，result 这个张量是计算节点“add”的第一个输出结果（编号从0开始）。

  • shape 属性描述了一个张量的维度信息。

例如样例中，shape=（2，）说明张量 result 是一个一维数组，数组长度为2。

  • type 属性指定了一个张量的唯一类型。Tensorflow 会对参与运算的所有张量进行类型检查，类型不匹配时会报错。
